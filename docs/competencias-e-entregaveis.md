## Resumo profissional

Desenvolvedor full-stack e especialista em automação de processos com IA.
Construo soluções de ponta a ponta — do app na mão do usuário ao backend, banco
de dados, integrações de pagamento e APIs — entregando produtos em produção, não
apenas protótipos. Foco em resolver o problema de negócio, não só escrever
código.

## APIs (integração, consumo e construção)

APIs são o ponto central de qualquer sistema moderno — é o que faz aplicativos,
sites, bancos de dados e serviços de terceiros conversarem entre si. Trabalho
nas três frentes:

### Consumo de APIs de terceiros

- Integração com serviços externos: gateways de pagamento (Stripe, Apple Pay),
  provedores de autenticação (Google, Apple), serviços de mensagem/push, e
  qualquer API REST documentada
- Autenticação de API: tokens Bearer/JWT, OAuth 2.0, chaves de API, headers e
  segredos
- Tratamento de erros, *retries*, *timeouts* e respostas resilientes a falhas

### Construção de APIs próprias

- Criação de endpoints REST (serverless / Edge Functions) para alimentar apps,
  sites e integrações
- Design de contrato de API: rotas, formatos de requisição/resposta, versão
- Validação de entrada, autenticação e autorização por papel
- Documentação para que outros sistemas consumam com facilidade

### Integração entre sistemas (o que mais gera valor)

- Conectar sistemas que hoje não se falam (ex.: app ↔ ERP ↔ banco de dados)
- **Webhooks**: reagir automaticamente a eventos (pagamento aprovado, novo
  cadastro, alteração de status) sem intervenção manual
- Sincronização de dados entre plataformas, eliminando digitação duplicada e
  planilhas manuais
- Automação de fluxos que disparam ações em cadeia entre serviços diferentes
